工厂模式
      不管哪种工厂模式，都是为了解耦

demo1 静态工厂（简单工厂） 容易理解，工厂是一个类，违背了开闭原则
      缺点：不符合开闭原则
           生产比较单一（例如当当只卖书）
      
demo2 工厂方法模式 ，符合开闭原则
      缺点:不停的增加类。
          生产比较单一（例如当当只卖书）
        
demo 抽象工厂模式，生产多个等级产品（例如当当只卖书 而 京东既卖鞋、裤子，又卖书）
